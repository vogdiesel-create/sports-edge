## Architecture Decisions
- ADR-001: Unders only (proven edge from backtesting)
- ADR-002: Flat $100 profit tracking (industry standard)
- ADR-003: Global picks, not per-user
- ADR-004: API key auth for API, JWT optional for web
- ADR-005: In-memory rate limiting (no Redis at this scale)
- ADR-006: Async SQLAlchemy + asyncpg
