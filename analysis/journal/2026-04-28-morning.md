## Root Cause: Early-Season Regression Flaw

The model uses `weight = IP / (IP + 60)` to regress K/9 toward league average (8.50).

For Ohtani with only 24 IP in 4 starts:
- Raw K/9: 9.38, Regressed: 8.75 (heavily pulled toward 8.50)
- IP projection: 5.61 (regressed from 6.0 toward 5.50 league avg)
- Matchup adj: 0.887 (Miami low-K team)
- Expected Ks: 4.84, then x0.86 bias = **4.16**
- FanDuel implies: **~7-8 expected Ks**

The model underestimates elite pitchers by 40-50% early season because it treats league average as the prior instead of using career/prior-season data.
