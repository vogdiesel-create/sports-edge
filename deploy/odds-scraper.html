How It Works

1. Devig each book's line -- remove the vig (juice) to get fair probabilities. For a 2-way market (over/under), fair_prob = implied_prob / (implied_over + implied_under).

2. Consensus fair probability -- average the devigged fair probs across all books offering this line. More books = more reliable consensus.

3. Find +EV -- if any book's offered odds imply a probability LOWER than the consensus fair probability, that's a +EV bet. You're getting better odds than the market says is fair.

4. Kelly sizing -- quarter-Kelly bankroll fraction for risk management.

Edge% = fair_prob - implied_prob (how much the book is off).
EV% = (fair_prob * payout) - (1 - fair_prob) (expected return per dollar).
